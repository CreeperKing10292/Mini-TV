Hardware page

Built on a Raspberry Pi

Mini TV is a plug-in device: the Pi boots directly into a fullscreen UI, reads a microSD local library , and outputs audio to an amplifier + speaker. This page captures the hardware direction and decisions.

Architecture Components

Architecture

The core idea is simple: Raspberry Pi → UI → media playback → audio amp → speaker. Keep ports serviceable (power + storage), keep heat managed, and keep the screen legible.

Compute
Pi 5 (8GB), running Debian Trixie with minimal startup times.
Display path
2" TFT Screen (128x160 Pixels) on SPI Interfacing.
Audio path
microSD → Amplifier → Speaker.

Key decisions

  • Power: 27V input with clean cable management
  • Thermals: equipped with a tiny but powerful CPU fan and heatsink to avoid any issues
  • Storage: microSD (16GB)
  • Controls: push buttons
